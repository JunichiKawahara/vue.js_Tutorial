section 16 算出プロパティで処理を含むデータを作成する

算出プロパティとは、任意に処理を含めることのできるデータです。

算出プロパティの使い方

算出プロパティは任意のデータを返す関数をcomputedオプションに定義します。

{{ width }} の半分は {{ halfWidth }} です。

算出プロパティを組み合わせて使用する

width = {{ width }}, height = {{ height }}

X: {{ halfPoint.x }}

Y: {{ halfPoint.y }}

ゲッターとセッター

{{ width }}
{{ halfWidth }}

算出プロパティのキャッシュ機能

算出プロパティとメソッドとの大きな違いとして、算出プロパティはリアクティブな依存データに基づき結果をキャッシュします。キャッシュを再構築するトリガになるのはリアクティブなデータのみです。
たとえば、次の算出プロパティが使っているMath.random()はリアクティブではないため、何度、使用しても同じ数字が返されます。
